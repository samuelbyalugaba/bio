## 🚀 About Me

Full-stack developer and systems builder with a focus on **enterprise-grade applications**, **AI-integrated platforms**, and **scalable architectures**. I operate at the intersection of clean code and real-world business impact — building systems that ship, scale, and solve actual problems.

Currently working across multiple ventures including **TSJ Diversified Group**, where I lead operational strategy, digital transformation, and technical architecture across a multi-sector portfolio.

- 🧠 **Core focus**: Full-stack web apps, AI-powered platforms, real-time systems, and enterprise SaaS
- 🏗️ **What I care about**: Clean architecture, measurable performance, maintainable codebases, and shipping fast
- 🌍 **Based in**: Dar es Salaam, Tanzania

- **Simple interfaces** → strong structure → measurable outcomes
- Build the **minimum** that ships, then iterate with instrumentation
- Prefer **proven architecture** for the core, and **sharp tools** at the edges (AI, real-time systems, UX polish)
